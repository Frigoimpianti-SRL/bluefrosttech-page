# Blue Frost Tech — Landing Page

Static landing page for Blue Frost Tech industrial refrigeration spare parts.

**Live site:** https://frigoimpianti-srl.github.io/bluefrosttech-page/

## Local preview

```bash
python3 -m http.server 8080
```

Open http://localhost:8080

## Update content

- Edit `index.html` for page content and layout
- Add or replace images in `photos/`
- Replace `logo.png` for logo changes

Push to `main` to publish via GitHub Pages.
